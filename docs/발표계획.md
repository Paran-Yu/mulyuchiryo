## 상세 내용

### 1. 영업 부장의 설명(4~5분)

- 배경
- 기획 의도
- 기능
- 특장점

### 2. 고객사 사장에게 판매(5~6분)

특장점을 시연을 통해서 다시 한번 확인시켜주는 느낌

1. 파라미터 상세 설정 가능!

- 어떠한 파라미터라도 다 대응!
- 당신의 공장에 맞춤형

2. 효율적인 알고리즘!

- 가장 최적의 vehicle을 할당
- 최적의 경로를 배정
- 교차로에서도 자체 알고리즘으로 충돌 회피

3. 사실적인 시뮬레이션!

- vehicle의 물리적인 모션 모사!
- 1초 단위까지 동작까지 놓치지 않음

4. 통계로 더 발전

- 다양한 parameter로 통계 기능 제공!
- 당신의 공장을 더 스마트하게 가능
